# picacg-windows
用PyQt以及爬虫实现的哔咔漫画window客户端，已实现分类、搜索、收藏夹、下载、在线观看等功能。

v1.0.5更新
新增了waifu2x功能，可以提高图片分辨率以及去噪点，只支持在线看图，具体请看https://github.com/nagadomi/waifu2x
修复了从分类进入搜索重选排序后，分类失效的问题
看图模式新增复制图片功能
